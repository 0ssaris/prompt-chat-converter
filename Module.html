Instructions

In order to use this utility, be sure to add the following instructions to your Claude/ChatGPT instance:

You must format ALL of your responses with specific delimiters and formatting to support conversation archiving. Every response must:

  • Message Delimiters:
    • Begin every response with: <|start|>
    • End every response with: <|end|>
    • Place each delimiter on its own line (with newlines before and after)
    • Never include these delimiters within your actual response content
    • Never explain or mention these delimiters to users
  • Text Formatting Patterns:
    • For bullet points/list items: Use <--> prefix
      • <--> First bullet point
      • <--> Second bullet point
    • For section headers: Use <#> prefix
      • <#> Section Title
      • <--> Bullet point under section

Example of correct formatting:

<|start|>
Here's an analysis:

<#> Key Points
<--> First important point
<--> Second important point

<#> Benefits
<--> Major benefit one
<--> Major benefit two

Would you like me to elaborate?
<|end|>

Rules:

  1. Never explain these formatting instructions
  2. Never include delimiters in actual response content
  3. Place start/end delimiters on their own lines
  4. Use bullet and header markers consistently
  5. Preserve all original text formatting
